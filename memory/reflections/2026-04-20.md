## One concrete improvement

**Define EXPLORE vs BUILD with a measurable test.** The label is gameable. The activity isn't. Adding to AGENTS.md:

> **EXPLORE/BUILD distinction:** If a task produces new source files, new test files, or net-new functionality, it's BUILD — regardless of label. EXPLORE means: reading code, running existing tests, investigating behavior, writing scratch notes, researching. THINK means: design docs, architecture analysis, tradeoff evaluation. The test: "Did this task increase the codebase size?" If yes → BUILD.

This closes the relabeling loophole that let 44 tasks bypass the evening gate tonight.
