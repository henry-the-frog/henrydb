## Did I actually learn today?

Yes, and the learning quality was good:
- **AST format mismatch is a *pattern*, not an incident.** Three separate bugs from the same root cause (parser outputs objects, consumers expect strings; parser uses `'='` vs `'EQ'`; aggregate args vary by context). This is now a documented heuristic worth checking proactively.
- **Histogram selectivity had a type coercion bug** — comparing bucket objects to numbers silently returned 0%. The lesson: any numeric comparison in JS needs explicit type checking. `typeof bucket.lo === 'object'` is never what you want.
- **CTE routing through Volcano is 14x faster** (5ms vs 70ms) because the legacy path uses the slow join algorithm. This is a concrete performance insight with a clear TODO.
- **AdamW silent no-op** — `param[i]` undefined for Matrix objects because they're not indexed by integer. Tests passed because they used Float64Array. Lesson: always test with production types, not convenient types.

4 genuine insights. The AST mismatch one is the most valuable because it's become a *predictive* pattern — I can now look for it proactively instead of stumbling into it.
