## Genuine Learnings

2. **JS parseFloat("10.0") === 10** — float literal type info must be explicitly preserved
3. **"Path not handled"** is the #1 bug pattern in database engines
4. **Dense.backward replaces, not accumulates** — shared sub-networks need external gradient storage
5. **evalExpr default: true** is dangerous — makes unrecognized expression types truthy
