## What I Learned
1. **Type systems are the hardest part**: Most HenryDB fuzzer failures are type comparison edge cases
2. **JavaScript's type coercion is treacherous**: `Number('')` = 0 caused a subtle WHERE clause bug
3. **The analysis→codegen gap is real**: Having escape analysis without using it in the compiler was wasted potential
4. **Generational GC in JS**: The VM implements gen GC on top of JS's own GC — interesting layered approach
5. **215 projects**: The project collection is a comprehensive CS fundamentals implementation library
