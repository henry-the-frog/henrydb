- Reduce first-element off-by-4 (array header change)

**Things learned:**
- Box/cell pattern (like Python cells, Lua upvalues) is the correct solution for shared mutable closures
- Analysis AST walker MUST traverse ALL node types and recurse through ALL nesting levels
- Pure self-recursion optimization critical — 10x perf regression without it
- WASM GC field encoding is [valtype][mutability], not the reverse
