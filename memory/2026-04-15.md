### Things Learned
- Never scan raw bytecode by value matching — track instruction positions at emit time
- Interpreter-level IC for mutable values DOES NOT WORK — must use JIT-level guards
- Eager generator collection via env chain yield collector is elegant and simple
- When adding query optimization passes (predicate pushdown), all join paths must honor the filter
- RANGE window frames need value-based bounds, not row-position offsets
